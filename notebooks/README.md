# Analysis Notebooks

This directory contains Jupyter notebooks for data analysis and strategy development.

## Notebooks

1. **01_exploratory_data_analysis.ipynb**
   - Explore market data structure
   - Analyze probability distributions
   - Feature correlations

2. **02_feature_importance.ipynb**
   - Visualize feature importance
   - Analyze feature stability over time
   - Identify most predictive features

3. **03_strategy_comparison.ipynb**
   - Compare individual strategies
   - Analyze ensemble performance
   - Strategy correlation analysis

4. **04_risk_analysis.ipynb**
   - Monte Carlo simulation results
   - Tail risk analysis
   - Drawdown analysis

## Setup

```bash
# Install Jupyter
pip install jupyter notebook

# Start Jupyter
jupyter notebook

# Or use JupyterLab
pip install jupyterlab
jupyter lab
```

## Usage

1. Ensure database is running and populated
2. Run pipelines to generate data
3. Open notebooks in Jupyter
4. Update database connection string if needed
